Updated with design tokens, anti-pattern tiers, and fixes from Claude + Codex adversarial review. Read through and approve or request changes.

What This Is

kurate.cloud is a taste capture system whose primary output is an LLM-consumable design brief. This document is both a taste filter (what to avoid) and a design system (what to build).

It emerged from 12 site reviews (91 likes, 28 dislikes, 12 commentaries) and was stress-tested by adversarial review from Claude and Codex.

ALWAYS Core Principles

Contrast is non-negotiable

WCAG AA minimum (4.5:1 body, 3:1 large). If brand colors fail contrast, adjust the shade, not the rule. FIXED

Color is semantic, not decorative FIXED

Every color communicates: brand identity, interaction state, data category, status, or hierarchy. Exception: marketing hero sections may use color expressively, but still intentionally.

No single element should BE the design

If you removed the element and the design collapsed, the element was doing too much.

Craft signals human curation FIXED

When assets unavailable: use honest placeholder boxes describing what's needed. Never fake curation with generic filler.

Constrain width intentionally

Content: 1280px max. Prose: 720px max. Forms: 480px max. NEW

Fewer words, larger type NEW

If a heading can be 4 words instead of 8, use 4.

Speak the user's language FIXED

Familiar ≠ generic. Familiar patterns + opinionated details = comfortable AND authored. Familiar patterns + defaults = template.

Progressive disclosure over complexity

Show basics, hint at depth. Surface essentials, let people drill in.

CONTEXT Available Moves

Tool / Dashboard

Dark default (#141414 or #1a1a2e). Chrome recedes. Violet for primary actions only. Blue for selection. Dense spacing (8-16px). Instant transitions. Skeleton loading.

Marketing (Apple reference)

Dark default (#000 or #000F3B). Dramatic type (48px+). Pill CTAs. Scroll animations. 64-96px section spacing. One message per page.

Documentation (Tailwind reference)

Navy dark (#0f172a). Sidebar + 720px prose. Syntax-highlighted code. Copy buttons.

Editorial

Light default (#faf9f6). Signifier serif headings. Earth tones. Gallery spacing.

NEVER Anti-Patterns

Hard Fails (auto-reject)

  • Body text below WCAG AA contrast
  • Body text at 35% opacity or lower
  • Cosmic/space gradient backgrounds
  • Rainbow/gradient text on headlines
  • "Revolutionary/innovative/all-in-one" copy
  • Fake company logos as social proof
  • Emoji as feature icons
  • No real content with no replacement markers
  • Designed as screenshot, not product

Strong Dislikes (fix these) EXPANDED

  • Glassmorphism/backdrop-blur as primary element
  • Weight 200, ultra-wide letter-spacing
  • Glowing gradient avatars
  • 16px+ radius everywhere uniformly
  • Gradient bars, gradient buttons, different gradient per card
  • Ghost-border social circles, wave emoji, "Now with AI!" pills
  • Cookie-cutter identical cards, decorative non-functional elements
  • Greyed trust logos, "10,000+" claims, "Everything you need" headers

Modern AI Slop NEW — 20+ patterns

  • Bento grids with pastel fills
  • Fake browser chrome around screenshots
  • Generic stat cards ("12.5k / 99.9% / +24%")
  • Dot-grid / mesh / blob backgrounds
  • 12 identical feature cards
  • Fake terminal snippets, decorative charts
  • Stock-smile placeholder avatars
  • AI sparkle/orbit motifs, noise overlays
  • 100vh heroes burying content
  • "Ready to get started?" gradient bands
  • Marquee logo strips, pricing table templates
  • Split hero with 15° mockup
  • Perfectly symmetrical layouts
  • Never invent fake brands, logos, or metrics

Review Checklist UPDATED

  1. Contrast — WCAG AA on all text + interactive elements
  2. Color — semantic, limited palette (5-7 functional colors)
  3. Craft — curated elements, honest placeholders
  4. Anti-patterns — hard fails = auto-reject
  5. Token compliance — font sizes, spacing, radius, shadows, motion NEW
  6. Layout — 1280px max, 720px prose, responsive breakpoints NEW
  7. Context match — right strategy for the deliverable
  8. Opinion — if you removed the logo, would you know whose system this is?
